https://www.w3schools.com/html/html_intro.asp HTML = Hyper Text Markup Language.
"Hypertext(하이퍼텍스트)"란 웹 페이지를 다른 페이지로 연결하는 링크라는 뜻이고
이렇게 꺽쇠 '<태그명>' 이런 녀석을 markup언어라고 합니다.


프로그래밍 언어라는 것은 제끼고, 그래서 하이퍼텍스트가 뭐냐고 물을 수 있습니다. 하이퍼텍스트는 웹 페이지를 다른 페이지로 연결하는 링크입니다.
여기를 누르면 네이버로 '연결'하는 것을 하이퍼 텍스트라고 합니다.

이것이 요소입니다.
각 요소들은 속성을 가질 수 있는데 이렇게 사용합니다.
속성이 반드시 항상 가져야 하는 것은 1. 속성사이에 공백. 2. 속성 이름 뒤에는 = 표시 3. 속성값의 앞뒤에 열고 닫는 ""인용부호 이렇게 된 녀석을 string이라고 합니다. 눈치채셨는지는 모르겠지만 위 아래 공간을 'br'이라는 태그로 띄우고 있습니다. 엔터로는 띄워지지가 않아요.
body라는 태그 안에 보여지는 모든 내용을 적습니다.
<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>
이곳을 눌러 홈으로 갑시다.
