함수

함수의 정의 함수 선언 및 표현식 등 배워보도록 하겠습니다.

1. 함수란?

  • 함수란 하나의 로직을 재실행 할 수 있도록 하는 것으로 코드의 재사용성을 높여줍니다.
  • 함수를 사용하려면 함수를 호출하고자 하는 범위 내에서 함수를 정의해야만 합니다.

2. 함수 선언 및 표현식, 호출

  • 함수는 다음과 같은 함수 키워드로 구성되어 있습니다.
    • 함수의 이름
    • 괄호 안에서 쉼표로 분리된 함수의 매개변수 목록
    • 중괄호 {} 안에서 함수를 정의하는 자바스크립트 표현

  • 예를 들어보겠습니다.

    function sumi( number ) {
    return number * number;
    }

    함수 sumi는 number라는 하나의 매개변수를 가집니다. 이 함수는 인수(number) 자체를 곱하여 반환하는 하나의 문장으로 구성되어있습니다.

    return문은 함수에 의해 반환된 값을 지정합니다.

    매개변수는 값으로 함수에 전달됩니다. 그러나 함수가 매개변수의 값을 바꾸더라도 이는 전역적으로 또는 함수를 호출하는 곳에는 반영되지 않습니다.

    ex.1.)
    var ex_a = 10;
    var ex_b = 20;
    function ex(ex_a, ex_b){
    return ex_a + ex_b;
    }
    ex(ex_a, ex_b);
    => 어떻게 나올까요? 답은 콘솔창에 ctrl+c/ctrl+v를 해보시길 바랍니다.

    함수 호출할 때는 ex(ex_a, ex_b)

    ex.2.)
    function myFunc(theObject){
    theObject.make = "Toyota"
    }
    var mycar = {make : "Honda", model : "Accord", year : 1998}; //mycar에 키와 값으로 변수 선언
    var x, y; // x y 변수 선언

    x = mycar.make // x의 값은 "Honda"입니다.

    myFunc(mycar); //함수 호출

    y = mycar.make; //y의 값은 "Toyota"입니다. make 속성은 myFunc에서 변경되었습니다.
    => 어떻게 나올까요? 답은 콘솔창에 ctrl+c/ctrl+v를 해보시길 바랍니다.

3. 재귀함수란?

  • 위에서 문제2번 풀어보셨나요? 재귀함수 문제였습니다.
  • 함수가 자신을 다시 호출하는 구조로 만들어진 함수입니다.
  • 재귀함수는 종료조건이 있어야 합니다. 종료조건을 설정해주지 않으면 무한 반복을 하게 됩니다.
  • 재귀함수로 작성이 되는 코드는 반복문으로도 작성할 수 있습니다.

  • ex.1.)
    function f(n){
    if(n <= 1) {
    return 1;
    }
    return n + f(n-1)
    }
    console.log(f(100))
    => 어떻게 나올까요? 답은 콘솔창에 ctrl+c/ctrl+v를 해보시길 바랍니다.

    여기서 사용된 for문 조건은 1부터 순회한다. 재귀함수의 경우는 100부터 순회한 다음 호출 스택에 쌓여 있는 값을 처리해 나가게 되는데 스택은 LIFO방식이기 때문에 이때 스택에 제일 마지막으로 반환된 1부터 100까지 순차적으로 더해서 값을 반환하게 된다. 곱하기 방식도 위와 같다.
