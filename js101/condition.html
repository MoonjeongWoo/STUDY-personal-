조건문 배우기

조건문은 말 그대로 조건이 있을때를 말하는 겁니다
얘를 들면 이라고 할수 있죠
혹시 주변에 만약에 친구 없나요?
그 친구는 아마 조건문을 마스터한 사람일 겁니다
조건문은 다음과 같습니다
내가 만약 외로울때면 누가 나를 위로해주지?
를 컴퓨터 언어로 해석을 해봅시다
일단 컴퓨터는 한글보다 영어를 좋아하기 때문에 영어로 번역을 해봅시다
어... 우리가 영어는 잘 못하지만 코딩은 잘 할수 있다 생각합니다
영어 만드는게 힘들다 그러면 한 단어만 생각해 봅시다!
조건문이 항상 뒤 따라오는 단어가 무엇일까요?
생각해 보면 항상 만약이라는 단어가 따라옵니다
만약 ~~~라면, 만약 ~~~일때, 등등 많습니다
항상 만약에 뒤에는 조건문이 붙죠
그래서 우리는 만약을 조건문으로 씁니다
근데 이제 우리모두가 아는 사실은 컴퓨터는 한글보다 영어중심입니다
그래서~ 우리는 만약을 영어로 써야 하겠죠
그래서 if를 쓰는 겁니다
그래서 내가 만약 배고프면 ~~ 을 컴퓨터 언어로 번역을 하면?
if(i == hungry) 라고 할수가 있겠죠!!
그리고 이런 if 구절을 if문이라고 우리는 고급지게 부른답니다
그리고 이 if 문을 쓰는 형식도 정해져 있습니다
if를 치고 나서 if 바로 뒤에 소괄호 () <- 얘를 넣어줍니다
그 다음에는 이 중괄호 {} <- 요놈을 써줍니다
그럼 최종적으로는 if(){} 이렇게 되겠네요
그렇다면 이제 조건을 써야겠죠?
조건은 저어어기 소괄호에다가 씁니다
만약에 내가 잘생겼다면을 if문으로 바꾸면?
if(me == 미남) 이렇게 되겠죠?
네 그런 다음에 조건만 붙이면 문장이 성립이 안되잖습니까?
컴퓨터도 문장을 다 만들어 줘야지 일을 합니다
그래서 우리는 결과를 저 중괄호 {} <- 여기에다가 쓰는 겁니다
자 다시 미남으로 돌아가서~ 내가 만약 미남이면 모델을 하겠다 를 컴퓨터 언어로 바꾸면?
if(me == 미남){do model}이라고 할수 있겟죠?
아유 if문 참 쉽다 그죠?
이제 우리가 할일은 저 소골호, 중괄호 에 있는 애들을 변수 처럼 만들어 주면 됩니다!
자 그러면 10초 과제르 한번 해보죠
i = 8이라고 변수선언을 해주세요
그리고 만약에 i가 10보다 작으면! 콘솔창에다가
메롱이라고 나오게 해주세요!
메롱이 나오시나요?
네 그러면 아주 기본적인 틀은 슬슬 잡히는 단계입니다!
그러면 이제 if문 말고 다른 조건문들도 보러가시죠!


else if

자 요놈은 뭐 하는 놈일까요? else if
요 else if 는 if 문의 또다른 버젼이라고 생각 하시면 됩니다
만약에 조건이 2개가 있어요
자 롤 라인 어디 가세요 물을때 보통 다 간다 하지 않고
탑이요, 미드요, 정글이요, 등등 이렇게 나오지 않겠습니까?
그러면 라인을 2개 간다고 치죠
저는 탑 미드를 갑니다! 라고 말하면 그러면 챔프는 뭐쓰세요?
하고 꼬리 질문이 나오겠죠? 이거를 else if 문을 써가지고 만들어 봅시다
저는 탑 할때는 베인을 하고 미드 할때는 야스오를 합니다 를 if문으로 바꿔보죠!
if (line == top) {champ = vayne}
else if(line == mid){ champ = yasuo}
이렇게 되겠죠?
네 여기서 else if 문을 잘 보시면! if문이랑 표기법이 완전히 똑같죠?
보시는 바와 같이 else if 문은 if문과 똑같이 씁니다! 단지 else 라는 구문이 들어가서 그렇지
그렇다면! 설명 보다는 직접 하는게 더 재미있고 배우기도 빨리 배우니!
i 라는 변수에 아까 8을 집어 넣었죠?
그러면 그 if 문에다가 else if 문을 추가 하죠!
근데 추가 하기전에! 조심 하셔야 될게 있습니다
else if 문은 if문이 먼저 있어야 해요! 없으면 else 라는 단어가 무의미 해지기 때문에
그리고 else if문은 if문 안에다가 쓰시면 안되요!
물론 쓸수야 있습니다!
근데 if문 안에 if문이 하나 더 들어가야지만 쓸수가 있겠죠?
여기서 if문 안에 if문이 하나 더 있는거를 이중조건문이라고 합니다!
다시 돌아와서! 아까 만들었던 if문 뒤에! else if문을 하나 만들죠!
하나 만드셨으면! 조건을 추가해 주세요!
i 가 문자열 "i" 보다 크면은 i변수를 출력해주세요! 라고 else if 문을 추가해주세요!
답이 뭐가 나오나요?
남들과 다른 정답이 나오면은 그건 다시 해보도록 합시다!


else


자 우리는 else if 문 까지 배웠죠?
그러면 else는 무엇일까요?
else는 꽤나 쉽습니다!
if문이나 else if문 조건식에서! 해당 되지 않는 애들!
을 else문 을 통해서 짬처리 하는 겁니다
한마디로 말하자면 짬통, 잔반 처리반이에요
예시로 귤 상자에 귤 있다고 합시다!
귤 상자에서 썩은것과, 벌레먹은것 들은 빼고 새 상자에 담는다
를 if문을 써서 만들면 어떻게 만들까요?
if (귤상자.귤 == 썩은것) { 빼다}
else if (귤상자.귤 == 벌레먹은것!){빼다}
else{ 새상자에 담는다}
이렇게 할 수가 있겠죠?
네 근데 else는 보는 것 처럼, 조건문을 안씁니다!
아까 제가 짬통이라고 하지 않았습니까? 나머지를 처리하는 전문반 같은 애들입니다
그래서 얘네들은 조건문을 안쓰고 그냥 결과만 보여주세요! 라고 컴퓨터에게 시키는 겁니다!
얼추 이해가 되셨나요?
이해가 되셨다면 한번 이제 전체적인 if문, else if문, else문을 이용해서
프로그램을 하나 만들도록 하죠!
그전에 담배 피고 오실분들은 피고 오시구요
저는 좀 있다가 나중에 만들도록 하겠습니다!
